# Fonts for the Sudoku board image

- **Fredoka** (`Fredoka-Variable.ttf`) — Google Fonts, SIL Open Font License (see `OFL.txt`).
  Rounded / bubbly look that fits the Bikini Bottom theme and includes digits 1–9.

## Why not KG Traditional Fractions?

KG Traditional Fractions (Kimberly Geswein) is a **fraction / math-symbol** font (½, ⅓, …),
not a full digit set for Sudoku. Personal-use free; commercial needs a paid license.

If you own a suitable school-handwriting TTF with digits 1–9, drop it in this folder —
the bot will try it after Fredoka.
